# Personal Site CMS

A minimal personal website built with Ruby on Rails, featuring a public-facing portfolio and a custom admin panel for content management.

## Overview

This project is designed as a small end-to-end application to showcase backend and full-stack development skills. It includes a public website presenting my experience and projects, along with an internal admin panel for managing content.

## Tech Stack

- Ruby 3.4.8
- Rails 8.1
- PostgreSQL
- Turbo (Hotwire)
- Bootstrap

## Features (MVP)

- Public personal website (About, Projects, Experience)
- Admin panel with authentication
- CRUD for projects and profile content
- Dynamic updates using Turbo
